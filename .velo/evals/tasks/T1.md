# T1 — Single-line typo fix

## Command

`/velo:task`

## Prompt

> Fix the typo "wokrflow" → "workflow" in WORKFLOW.md line 14.

## Setup

Plant the typo first. From the repo root, run:

```
sed -i.bak '14s/workflow/wokrflow/' WORKFLOW.md && rm WORKFLOW.md.bak
```

Verify the typo exists at line 14 before running the task.

## Expected behavior

- One agent spawned (or zero — Velo edits inline since it's a single-character fix on a known line).
- No spec-checker, no review phase, no PM, no TL.
- Edit applied, verified, committed in under 2 minutes wall time and under 5k tokens.

## Anti-patterns to flag

- Spawning spec-checker for a typo.
- Spawning more than one agent.
- Triggering the full review chain.
- Reading more than 3 files.

## Scoring notes

Agent efficiency and cycle discipline are the dominant signals. Outcome correctness is binary (typo fixed or not). Scope discipline matters if the agent edits anything beyond line 14.
